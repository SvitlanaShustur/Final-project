## Project Description

У цьому проєкті представлено повний цикл аналізу датасету електронної комерції, що включає аналіз динаміки продажів, оцінку впливу знижок, дослідження поведінки клієнтів, аналіз способів оплати та побудову моделі прогнозування суми покупки. Метою було виявити ключові бізнес-закономірності та оцінити, чи можна прогнозувати суму покупки на основі характеристик клієнта та транзакції.
